# ML Verification Result Predictor

A machine learning web app built with Streamlit that predicts verification results using Logistic Regression and Decision Tree models.

## 🚀 Live Demo
[Click here to view the app](https://manojkumar9676-ml-streamlit-app.streamlit.app)

## 📌 Features
- Loads real-world dataset automatically
- Choose between Logistic Regression and Decision Tree
- Displays model accuracy
- Enter custom input and get instant predictions

## 🛠️ Tech Stack
- Python
- Streamlit
- Scikit-learn
- Pandas

## ⚙️ Run Locally

1. Clone the repo
   git clone git@github.com:manojkumar9676/ml-streamlit-app.git

2. Install dependencies
   pip install -r requirements.txt

3. Run the app
   streamlit run app.py
